Dataflow

Profiled class IRI(s) dcat:Catalog
IRI https://statdcat-ap/#Dataflow
Label Dataflow
Definition A catalogue or repository that hosts the Datasets or Data Services being described.
Hierarchy
  • profiles class profile Catalogue (dcat-ap:Catalogue)
    Definition: A catalogue or repository that hosts the Datasets or Data Services being described.
    • profiles class profile Catalog (dcat-dap:Catalog)
      Definition: A curated collection of metadata about resources.
      • profiles class Catalog (dcat:Catalog)
        Definition: A curated collection of metadata about resources (e.g., datasets and data services in the context of a data catalog).
        • specializes class Dataset (dcat:Dataset)
          Definition: A collection of data, published or curated by a single source, and available for access or download in one or more representations.
          • specializes class Catalogued resource (dcat:Resource)
            Definition: Resource published or curated by a single agent.
      • specializes class profile Dataset (dcat-dap:Dataset)
        • profiles class Dataset (dcat:Dataset)
          Definition: A collection of data, published or curated by a single source, and available for access or download in one or more representations.
          • specializes class Catalogued resource (dcat:Resource)
            Definition: Resource published or curated by a single agent.
        • specializes class profile Cataloged Resource (dcat-dap:CatalogedResource)
          • profiles class Catalogued resource (dcat:Resource)
            Definition: Resource published or curated by a single agent.
Usage note A Web-based data catalog is typically represented as a single instance of this class. Datasets and data services are examples of resources in the context of a data catalog.

Range Dataset [1..*]
Hierarchy
  • profiles property profile dataset (dcat-ap:Catalogue.dataset)
    Definition: A Dataset that is part of the Catalogue.
    • profiles property profile dataset (dcat-dap:Catalog.dataset)
      • profiles property dataset (dcat:dataset)
        Definition: A dataset that is listed in the catalog.
        • specializes property resource (dcat:resource)
          Definition: A resource that is listed in the catalog.
          • specializes property Has Part (dcterms:hasPart)
            Definition: A related resource that is included either physically or logically in the described resource.
            • specializes property Relation (dcterms:relation)
              Definition: A related resource.
Usage note As empty Catalogues are usually indications of problems, this property should be combined with the property service to implement an empty Catalogue check.
